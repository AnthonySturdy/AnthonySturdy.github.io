ABOUT

Hi, I'm Anthony. I'm currently working at Unity Technologies as a Sustained Engineer Intern, as part of the sandwich year of my university course.


The course I'm studying is BSc Computer Games Programming at Staffordshire University, my coursework primarily involves using C++ to create games, game engines and frameworks using DirectX, OpenGL and SDL. We also use C# for networked applications and multithreading, I have additional experience with C# from using Unity for previous college coursework and for occasional game jams. My limited web development experience comes from my first year at University, and creating websites such as this portfolio.
